# AEA Website Updates — March 2026

Implementation specs and mockups for planned aea.net website changes.

## View the Mockups

**Enable GitHub Pages** on this repo (Settings → Pages → Source: main branch, root), then visit:

```
https://YOUR-USERNAME.github.io/aea-website-updates/
```

This gives you a clickable dashboard linking to every mockup and spec document.

**Or just open locally** — clone the repo and open `index.html` in your browser.
