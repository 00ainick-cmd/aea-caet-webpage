# Workforce Folder Setup

## Overview
Create a new `/workforce/` directory on aea.net to host military transition resources with permanent, shareable URLs.

## Folder Structure
```
aea.net/workforce/
├── careerplaybook.pdf      → Military Avionics Career Playbook
├── hiringtoolkit.pdf       → Military Hiring Toolkit
└── (future resources)
```

## URLs
| Resource | URL |
|----------|-----|
| Career Playbook | `aea.net/workforce/careerplaybook.pdf` |
| Hiring Toolkit | `aea.net/workforce/hiringtoolkit.pdf` |

## Notes
- Actual filenames on the server can vary — what matters is the URL path
- No index page needed initially (direct PDF links), but consider adding one later as a landing page for workforce development resources
- These URLs will be used in printed materials, QR codes, LinkedIn posts, and the LOA Sustainer Series — they need to be permanent
- Future additions may include additional guides, the NextOp webinar recording, and other military transition resources
